api REST API Example
Python
import requests
 
# Get latest BTCUSD price
response = requests.get(
  'https://api.yourexchange.com/v1/ticker',
  params={'symbol': 'BTCUSD'})
 
print(response.json())
stream WebSocket Example
JavaScript
const ws = new WebSocket('wss://api.yourexchange.com/v1/stream');
 
// Handle incoming data
ws.on('message', (data) => {
  console.log(JSON.parse(data));
});
